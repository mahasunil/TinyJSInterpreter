#Tiny Js Interpreter In C++

**Tiny Js Interperator** is minimal javascript interpreter implemented in C++ for educational and learning purpose.

##Overview
 This project demonstrates the basic components of a Javascript interpreter
**lexical analysis:** Converts source code into tokens.  
**parsing:** Builds abstract syntax tree (AST) from tokens.
**Interpreting:** Evaluates the AST and excutes code.

##Features
-Parses and evaluates simple arithmetic expression: '+', '-', '*', '/'.
-Supports parenthesis for grouping.
-Implements a basic Abstract Syntax Tree (AST).
